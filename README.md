# eSportskiCentar
eSportskiCentar is a web-based application designed primarily for employees and management of sports centers. The system manages scheduling of sports events based on available or user-defined time slots. It supports four user roles: system administrator, sports center manager, staff member, and client.

The application is mainly focused on internal organization, while external users (clients) have limited functionality, including viewing event schedules and submitting booking requests. This feature is especially useful for both event audiences and sports organizations.

The administrator has a wide range of responsibilities, including managing all system data and user accounts. They can create and manage sports facilities (such as halls, courts, or fields) by defining parameters like dimensions, equipment, surface type, and capacity. Based on these attributes, the system automatically determines suitable event types, reducing potential errors.

The sports center manager plays a key role in organizing events within a specific facility. They can create and manage events, which typically represent matches scheduled at specific times. The system also supports competition registration, simplifying data entry and reporting.

An important interaction exists between managers and staff members, where managers can assign tasks through notifications with defined deadlines. Tasks related to events are automatically assigned based on predefined rules for each event type.
